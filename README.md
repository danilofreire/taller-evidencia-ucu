# Taller: Políticas públicas basadas en evidencia

Sitio del taller "Políticas públicas basadas en evidencia: lecciones desde la investigación aplicada en América Latina", Universidad Católica del Uruguay.

## Estructura

- `index.qmd` — inicio
- `programa.qmd` — programa detallado por bloque
- `materiales.qmd` — diapositivas, datos y código
- `recursos.qmd` — lecturas, herramientas e instalación
- `diapositivas/` — decks por bloque (en preparación)
- `theme.scss` — tema visual del sitio
- `_quarto.yml` — configuración del sitio

## Construir el sitio

Requiere [Quarto](https://quarto.org).

Previsualizar con recarga automática:

    quarto preview

Renderizar el sitio completo a `docs/`:

    quarto render
